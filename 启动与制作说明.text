以mac电脑为例，windows类似

1.项目制作中，搭建前后端重要步骤：
	(1)安装了node（node.js),安装npm,安装全局的express
	(2)到目标路径，用express建立工程。
	命令行输入： express * -e (注意如果不输入-e默认会以jade engine，-e为ejs engine)
	(3)进入项目导入npm，
	命令行输入： npm install
	(4)安装监听的小工具，supervisor
	命令行输入： npm install -g supervisor
	(5)监听和启动服务器，
	命令行输入： supervisor bin/www
	在浏览器中输入： localhost:3000


2.启动说明
	到项目目录下，
	命令行中输入： supervisor bin/www
	在浏览器中输入： localhost:3000
